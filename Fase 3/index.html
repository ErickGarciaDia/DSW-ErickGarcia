Etiquetas comunes o basicas del HTML con sus Atributos y valores

Como ya se sabe toda etiqueta termina con la misma etiqueta solo que dentro de la misma se coloca el caracter / y el nombre de la etiqueta a cerrar. Muchas de estas etiquetas tienen sus propios Atributos y valores, ademas permite aplicar mejorar a los textos de la pagina web o a la misma pagina. Algunos atributos y valores se pueden utilizar en sin fin de etiquetas pero otros como las etiquetas para tablas o formularios tienen unicos atributos.

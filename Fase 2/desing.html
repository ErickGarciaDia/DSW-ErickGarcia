Proposito del sitio

Etiquetas comunes o basicas del HTML con sus Atributos y valores

Como ya se sabe toda estiqueta termina con la misma etiqueta solo que dentro de la misma se coloca el caracter / y el nombre de la etiqueta a cerrar. Muchas de estas etiquetas tienen sus propios Atributos y valores, ademas permite aplicar mejorar a los textos de la pagina web o a la misma pagina. Acontinuacion se presentan las etiquetas mas utilizadas y comunes:

  • b ó strong: Aplica negrita al texto incluido entre las etiquetas.
  • i ó em: Aplica cursiva al texto incluido entre las estiquetas.
  • u: Aplica subrayado al tecto incluido entre las etiquetas.
  • font: Delimita un texto con un formato de funente determinado. Esta etiqueta tiene Atributos o propiedades como son:
    • font size="X": Indica el tamaño del texto
    • font color="#RRGGBB": Define el color del texto donde cada letra representa dentro de las comillas un valor que indica el color.
    • font fase="arial": Determina el tipo de fuente del texto, es decir su tipografia.

      • Estas etiquetas puede incluirse dentro de la etiqueta font simultaneamente y para cerrarlas se utiliza /font.

  • h1, h2..., h6: Indica 6 niveles de formato de encabezados siendo h1 el mas grande y h6 el mas pequeño.
  • br: Coloca un salto de linea donde el usuario lo considere mejor
  • p: Delimita un parrafo de texto, con el atributo aling se pueden agregar los siguientes valores
    • center: Alineacion centrada
    • left: Alineacion a la Izquierda
    • rigth: Alineacion a la Derecha
    • justify: Alineacion justificada
  • hr: Inserta una linea horinzotal que sirve para dividir secciones o como lo prefiera el usuario.
  • div: se utiliza para delimitar una seccion dentro del documento, de forma que agrupe un numero de elementos para luego añadirle un estilo determinado
  • ul: Hace una lista no numerada, es decir, sin un orden establecido
  • ol: Hace una lista numerada, es decir, que tienen un orden establecido (1 a infinito).
  • li: Es un elemento(s) de la lista definida en las 2 anteriores etiquetas.
  • img src="ubicacion de la imagen": Inserta una imagen que se encuentra en la ruta indicada dentro de las comillas. Tambien se puede incluir una serie de Atributos y propiedades
    • border="x": Establece un borde de tamaño de x pixeles entrono a la imagen.
    • heigth="x": Establece una altura x a la imagen.
    • width="x": Establece un ancho x a la imagen.
    • alt="texto explicativo": Se muestra un texto al pasar el raton sobre la imagen.
    • aling="valores": Determina la alineacion de la imagen con los siguientes valores dentro de las comillas
      • bottom
      • middle
      • top
      • left
      • rigth
    • hspace="x": Hace un espacio horizontal en pixeles entre la imagen y el texto.
    • vspace="y": Hace un espacio vertical en pixeles entre la imagen y el texto.
  • table: Define donde poner una tabla. Con los Atributos width, heigth, border se puede aplicar a esta etiqueta (table) con las mismas funciones anteriores que en la etiqueta img, video y audio, pero los 2 siguientes Atributos son exclusivos de table
    • cellspassing="x": Define el espacio en pixeles entre las celdas.
    • cellpadding="x": Define el espacio en pixeles entre el borde de la tabla y el texto.
  • tr: Indica el comienzo de cada una de las filas.
  • td: Indica el comienzo de cada una de las columnas o celdas de la tabla.
  • a href="link de la pagina web"> nombre del enlace: la etiqueta a permite la creación de enlaces, con el atribto href="enlace" permite colocar el enlace talcual que quiere el usuario y junto con el nonbre que quiera darle dentro de la pagina web
  • a href="Correo electronico"> nombre del enlace: Es igual que la anterior etiqueta solo que en vez de un enlace web en especifico, el usuario puede poner un correo electronico y definirle un nombre que aparezca en la pagina web
  • form: Indica el inicio de un formulario
  • label: Define una etiqueta de texto dentro del formulario.
  • input: Indica el principio de algun control de entrada de datos, puede ser de varios tipos segun el atributo type:
    • text: Entrada de texto.
    • password: Tambien introduce un texto pero cuyos caracteres se miran en asteriscos en la pagina web. De gran utilidad para colocar contraseñas.
    • checkbox: Es una casilla de verificacion.
    • radio: Elementos de selección entre varias opciones tipo radio
    • sumbit: Boton para enviar el formulario
    • image: Boton para borrar todo el contenido de datos del formulario
    • reset: Boton para borrar el contenido de datos de usuario
    • hidden: Es un campo oculto que envia una informacion que no muestra el formulario
    • file: Permite adjuntar un archivo desde el ordenador.
  • select: Delimita el principio de una lista desplegable de opciones, cada opcion se indica mediante la etiqueta option.
  • textarea: Permite introducir un texto largo.

Atributos mas comunes

Los Atributos mas utilizados dentro de muchas etiquetas son:

  • id: Proporciona un ientificador unico en todo el documento.
  • class: proporciona una manera de clasificar elementos similares por lo que se utiliza con propositos de presentacion y de semantica.
  • style: Este se puede usar en cualquier elemento de HTML y permite definir estilos a estos.
  • title: se suele usar para definir el titulo de la pagina en la barra de titulo del navegador web.
